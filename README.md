# Quadratization in Pseudo-Boolean Optimization and Adiabatic Quantum Computing
A far-reaching literature review on the quadratization of pseudo-Boolean functions.

## Rules for Contributing
1. Write each sentence on a new line. This makes git version control easier to use.
2. Add references to the k-local-review.bib file, and also to the group at https://www.mendeley.com/groups/9986881/k-local-quadratization/
3. Use the \examplesec sections commands to maintain consistent style.
